Characterization of non-linear systems through high-order statistics

  • Information Theory for Signal Analysis (ITSA) .
    • This Matlab toolbox illustrates the use of auto-mutual information (mutual information between subsets of the same signal) and entropy rate as powerful tools to assess refined dependencies of any order in signal temporal dynamics.

  • Two-dimensional Structure functions for direction-dependent multiscale Image Texture Analysis (TSITA) .
    • This Python toolbox illustrates the use of second and high order structure functions as powerful tools to characterize image textures.

Super Resolution of Remote Sensing Land Surface Temperature Images

  • Thermal Unmixing library for python (ThUnmpy).
    • This Python toolbox contains several sharpening methods for super resolution of Land Surface Temperature thanks to data-fusion with higher resolution features in the shortwave optical spectral domain.
